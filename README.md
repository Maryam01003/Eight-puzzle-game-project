# 8-Puzzle Game

Interactive sliding puzzle game built with Python Tkinter. Solve 3x3 puzzles manually or with AI (BFS solver). Tracks moves/time, high scores saved to JSON.

## Features
- Colorful tile grid with click-to-move
- Scramble for new games
- Auto-solver with animation
- High scores with name entry
- Undo moves
- Stats (moves, time)

## How to Run
1. `python eight_puzzle.py`
2. Data saved in `data/scores.json`
